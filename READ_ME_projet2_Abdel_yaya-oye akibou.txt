Projet_2 MASB ("Methodes pour l’Analyse de Sequences en Bioinformatique"): YAYA-OYE AKIBOU Abdel M1 MISO

DESCRIPTION:
	This program makes it possible to align reads on a genome according to a seed and extend strategy. 
	This is a semi-global alignment.

	Out : File containing the different reads aligned to the genome according to the line format below
	read_id\tposition\tscore\tidentity\tsequence\n

FUNCTIONS:
	### project2_align_semiglobal.py ###
	- extract_kmers_from_fastq_file(filename, k)
	- extract_sequences_from_fasta_file(filename)
	- generate_suffix_table(sequence)
	- search_kmers_in_genome(fastq_file, fasta_file, k)
	- align_reads_to_genome(genome_file, reads_file, output_file, k, threshold)
	- main()

DATA:
	filepath 	= filename.fna      (coronavirus.fna, gripeA.fna, HIV.fna, rhinovirus.fna)
	filereads 	= filename.fastq.gz	(SRR10971381_1.fastq.gz, SRR10971381_2.fastq.gz)



HOW TO USE IT:
	On the shell (terminale), use the "cd" command to go to the "Porjet2_V2_MASBYAYA-OYE AKIBOU Abdel" folder 
	and then, write according to the following model: 
	$ python3 project2_align_semiglobal.py -g file.fasta -r file.fastq -o outfile.txt -k 11=default -t 90=default
    
    Required arguments :
    -------------------
        - -genome (-g) ==> filepath = path to a file in FASTA format (.fasta) containing the reference genome
        - -reads (-r) ==> filereads = path to one or multiple FASTQ.gz containing the reads
        - -outfile(-o) ==> outfile = Name of a outfile
      
        
    Optionnal argument :
    --------------------
        - -k : kmersize = k-mer size (? by default)
       - - t : threshold = identity (? by default)
